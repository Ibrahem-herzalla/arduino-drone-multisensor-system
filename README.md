# Arduino-Based Multi-Sensor Drone System

## Overview
Embedded system mounted on a drone for real-time environmental data collection (temperature, humidity, distance, color), transmitted via Bluetooth to a mobile application.

## Features
- Multi-sensor data acquisition  
- Bluetooth communication  
- Mobile monitoring interface  
- LED feedback system  
- Full hardware-software integration  

## My Role
- Developed the embedded logic in Arduino C
- Integrated sensors and Bluetooth communication
- Built the MIT App Inventor mobile interface
- Tested the system on a drone-mounted platform
